# Field Dictionary

- `company_name`: 企业名称
- `plan_name`: 预案名称
- `plan_version`: 预案版本号
- `plan_issue_date`: 预案签署或实施日期
- `plan_code`: 预案编号
- `compiler_org`: 编制单位
- `service_org`: 技术服务机构
- `organization_code`: 统一社会信用代码或机构代码
- `legal_representative`: 法定代表人
- `legal_phone`: 法定代表人联系电话
- `contact_person`: 联系人
- `contact_phone`: 联系人联系电话
- `fax`: 传真
- `email`: 电子邮箱
- `address`: 企业地址
- `longitude_latitude`: 经纬度
- `industry`: 所属行业
- `region`: 所在地区
- `risk_level`: 风险级别
- `risk_materials`: 风险物质及储量
- `risk_sources`: 环境风险源
- `environmental_receptors`: 周边环境风险受体
- `pollutant_discharge`: 污染物排放与治理措施
- `emergency_org_structure`: 应急组织体系
- `emergency_contacts`: 应急联系人
- `emergency_resources`: 应急资源
- `emergency_monitoring`: 应急监测要求
- `warning_mechanism`: 预警机制
- `response_process`: 应急响应流程
- `onsite_disposal`: 现场处置措施
- `post_disposal`: 后期处置
- `training_plan`: 培训计划
- `drill_plan`: 演练计划
- `review_comments`: 专家评审意见
- `public_comments`: 征求意见及采纳情况
